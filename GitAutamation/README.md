# GitAutomation

Automação para projetos Git utilizando Git Hooks e GitHub Desktop.

## Recursos

- Atualização automática do CHANGELOG.md
- Compatível com GitHub Desktop
- Compatível com VS Code
- Utiliza Conventional Commits
- Instalação com um clique
- Reutilizável em qualquer repositório

## Estrutura

```
GitAutomation/
│
├── cliff.toml
├── scripts/
├── hooks/
└── docs/
```
